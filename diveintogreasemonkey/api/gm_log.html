名称

GM_log — 记录日志到错误控制台

大纲

function GM_log
                           (
                        message
                        , 
                        level
                        );
                     

                  

               

               
 
                  
                  
描述

                  
                  

                     GM_log 在错误控制台中输入信息。主要用于调试。level 是可选的，默认值是0。其有效的值有：0 - 信息、1 - 警告、2 - 错误。 
                  

                  
               

               
 
                  
                  
历史

                  
                  

                     GM_log 由 Greasemonkey 0.3 引入。
