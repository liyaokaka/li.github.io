名称

GM_addStyle — 给页面添加 CSS 样式

大纲

function GM_addStyle
                           (
                        css
                        );
                     

                  

               

               
 
                  
                  
描述

                  
                  

                     GM_addStyle 给页面添加样式。创建一个新的 <style> 元素，在其中加入给定的 CSS，然后插入到 <head> 中。
                  

                  
               

               
 
                  
                  
历史

                  
                  

                     GM_addStyle 由 Greasemonkey 0.6 引入。
