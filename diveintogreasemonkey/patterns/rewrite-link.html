4.19. 改写链接

如果获取到页面上的链接，可以通过设置它的 href 属性来改写链接。

例 4.23. 在链接末尾添加查询参数

假设有个链接的 id 为 "article"。 

var a = document.getElementById('article');
if (a.href.match(/\?/i)) {
	// 链接已经包含其它参数，所以添加 "&amp;printer=1"
	a.href += '&amp;printer=1';
} else {
	// 连接没有包含任何参数，所以添加 "?printer=1"
	a.href += '?printer=1';
}
