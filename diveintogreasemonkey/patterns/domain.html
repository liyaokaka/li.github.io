4.1. 在域名以及它所有子域名上执行用户脚本

对于许多网站，无论是否有 www. 前缀，访问网站都是等效的。如果要为这样的站点写用户脚本，需要能匹配这两种地址。

例 4.1. 匹配域名和它所有子域名的元数据标签

// ==UserScript==
// @include http://example.com/*
// @include http://*.example.com/*
// ==/UserScript==
