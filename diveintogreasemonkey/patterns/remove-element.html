4.9. 删除元素

使用 removeChild 函数，您可以使 Greasemonkey 一下子删除页面中完整的一块。

例 4.12. 删除广告侧边栏

假设有个元素 ID 为 "ads"。 

var adSidebar = document.getElementById('ads');
if (adSidebar) {
	adSidebar.parentNode.removeChild(adSidebar);
}

用removeChild 移除一个元素也会删除它里面的所有内容。例如您移除一个 <table> 元素，此时也会将表格单元格(<td> 元素)一起移除。

如果要移除广告，比起写用户脚本来，更容易的办法是安装 AdBlock 然后导入 最新的过滤列表。
