4.7. 在元素前插入内容

如果已找到某个元素(不论用何种方法)，您想在它前面插入额外的内容。您可以使用 insertBefore 函数。

例 4.10. 在主内容前插入 <hr>

假设有个元素的 ID 为 "main"。 

var main, newElement;
main = document.getElementById('main');
if (main) {
	newElement = document.createElement('hr');
	main.parentNode.insertBefore(newElement, main);
}
